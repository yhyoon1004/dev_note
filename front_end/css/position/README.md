# CSS Position 속성
> position : 자신을 어떻게 배치할 것 인가?

- `position-relative`  
  기본적인 html 요소 `블럭`, `인라인` 배치 형식  
  자신을 문서 흐름에서 벗어나지 않으면서, 요소의 원래 위치를 기준으로 상하좌우로 이동
- `position-absolute`  
  절대 좌표 기준으로 배치  
  자신에게 가장 가까운 `position`이 `relative`, `absolute`, `fixed`, `sticky`로  
  설정된 조상 요소를 기준으로 위치를 지정할 때 사용하는 속성
